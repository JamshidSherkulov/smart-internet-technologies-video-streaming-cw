# Smart Internet Technologies Coursework
## DASH Video Streaming Experiment

Student ID: 4223542 
Module: Smart Internet Technologies  
University: London South Bank University

## Project Goal

This project implements an end-to-end DASH video streaming experiment
to analyse the impact of network artefacts on video Quality of Experience (QoE).

The experiment includes:

- Video transcoding using FFmpeg
- DASH manifest generation
- Web server video streaming
- Client playback
- Network traffic simulation using Linux Traffic Control
- Performance evaluation using MOS

## Project Structure

videos-original → downloaded HD videos  
videos-transcoded → transcoded videos  
dash-output → DASH manifest and segments  
scripts → ffmpeg, tc and iperf commands  
results → experiment results and MOS scores  
screenshots → screenshots used in the report
